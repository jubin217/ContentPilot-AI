# Building LinkedIn Paragraph Creator

This extension is built using React, TypeScript, TailwindCSS, and Vite.

## Requirements
- Node.js (v18 or higher)
- NPM

## Steps to Build from Source
1. Install development dependencies:
   ```bash
   npm install
   ```
2. Compile and package the extension:
   ```bash
   node build.js
   ```

The compiled and built assets will be generated inside the `dist/` directory, which matches the contents of the uploaded extension package.
